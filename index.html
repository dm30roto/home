  • Recon Rally Team Leader/Mobile Front-End Developer

    I'm working on a team of RPI students to build a mobile event-based social application that incorporates NFC technology. These Rallies can be time specific such as sporting events and concerts or can be taken at the user's convince such as a campus tour. Each Rally has a set number of checkpoints that the user can find and tag, and the Rally is complete when all checkpoints for that Rally are found. Each checkpoint will provide information about that Rally or where that checkpoint is, and provide access for more information. We believe this application will help promote users to discover and attend more events and to get users more involved within their communities. The application should be in the Google Play store by December 2012 (search for Recon Rally).
